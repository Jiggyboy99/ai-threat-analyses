## 🧭 Two kinds of analysis

**Threat Analyses** dissect a real-world AI security incident — what happened, how the attack worked, how it maps to threat frameworks, the impact, and the defenses.

**Risk Assessments** evaluate a specific AI model *before* it's trusted in production — scoring risks across data provenance, licensing, bias, the inheritance problem, and security, then issuing a clear verdict with recommended controls.

Both follow a consistent, professional structure so each piece reads like real practitioner work, not a tutorial.
